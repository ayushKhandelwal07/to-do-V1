# To-Do -V1

Implementation of Todo web-application

## Used techstack
## Backend
 * Express for serverside logic
 * Zod for validation
 * Mongoose for mongo Connection
 * MongoDB as a database

## Frontend
 * React

## Operations Support
Routes and operations
```javascript

# create todo
app.post('/todo')

# returns 'todo'
app.get('/todo')

# update todo
app.put('completed')
```
